使用说明

  1. 使用源账号（你要克隆的主账号）登录
  2. 系统会显示账号的详细信息
  3. 修改货币：选择操作类型并输入数值，然后确认修改
  4. 克隆操作：选择操作类型：修改当前账号ID 或 克隆到新账号
  5. 输入自定义Local ID
  6. 如选择克隆到新账号，输入目标账号凭据
  7. 重要：操作过程中不要关闭浏览器！
提示：
  • 修改货币：通常需要10-30秒完成
  • 克隆账号：时间取决于车辆数量，通常需要2-5分钟
  • 修改ID：大约需要1-2分钟完成
